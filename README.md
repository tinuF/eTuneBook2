# Introduction

Migration of eTuneBook from Angular 1 to 2. 

Project started with angular2-seed by Minko Gechev

# How to start

```bash
git clone https://github.com/tinuF/eTuneBook2.git
cd eTuneBook2
npm install
# If you don't have gulp already installed
npm install -g gulp
# dev
gulp serve.dev
# prod
gulp serve.prod
```
